You can connect any canvas to a GitHub repository, clone it locally, edit widget code in your own editor, and push changes that automatically update the live canvas.

## Connecting to GitHub

1. Open the canvas you want to work on
2. Use the Connect Github option in the profile dropdown to connect a repository
3. DeepSpace creates a repo (or connects to an existing one) and grants you push access
4. Accept the collaboration invitation sent to your email
5. Clone the repository locally
6. Read the README.md file in your repository -- it includes detailed steps for setting up and pulling/pushing commands

Once connected, a webhook is set up so that every `git push` triggers an automatic rebuild of any changed widgets.

## Tips

<Tip>
**Pull before you push.** If someone else (or the AI) has been editing widgets on the canvas while you're working locally, `git pull` first to get the latest `canvas-state.json`. The post-merge hook will unpack the changes into your local files.
</Tip>

<Warning>
**Don't edit `main.tsx` under widget source code** — this file is the platform wiring that handles auth, providers, and context detection. It gets overwritten on builds. Put your code in `App.tsx` and other files.
</Warning>
