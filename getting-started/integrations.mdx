DeepSpace comes with dozens of built-in integrations so your widgets can connect to external services out of the box. No API keys, no authentication headaches — just describe what you want and the AI wires it up.

## How to Use Them

You don't need to configure anything. When you ask the AI to build a widget, just describe what you need and it will connect the right services automatically:

- "Show my Google Calendar events for this week"
- "Create a widget that generates images from text prompts"
- "Build a research tool that searches academic papers"
- "Track the latest Premier League standings"
- "Search for flights to Tokyo next month"

## Approval for API Calls

Some integrations use credits (like text generation or image creation). When a widget tries to use one of these for the first time, you'll see a quick approval prompt asking if you want to allow it. You can:

- **Allow Once** — Approve just this call
- **Allow Always** — Auto-approve all future calls from this widget (for the rest of your session)
- **Block** — Deny the call

This keeps you in control of which widgets can make API calls on your behalf. Free and read-only endpoints (like searching or fetching public data) are approved automatically.

DeepSpace handles all the API complexity and authentication for you. When your widget needs data from an integration, it makes a simple request and gets data back — no boilerplate, no setup.
